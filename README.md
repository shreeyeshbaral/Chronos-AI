# 🧠 Chronos AI

> **AI-Powered Productivity Workspace**

Chronos AI is an intelligent productivity platform that helps users organize tasks, analyze productivity, track progress using AI, and work in an adaptive environment designed for different cognitive and sensory preferences.

Built using **React**, **Firebase**, **Google Gemini AI**, and deployed on **Google Cloud (Firebase Hosting)**, Chronos AI combines intelligent task management, analytics, accessibility, and AI assistance into one modern productivity workspace.

# 📌 Problem Statement

Managing tasks efficiently is challenging when deadlines, priorities, productivity, and accessibility needs are spread across multiple tools.

Chronos AI solves this by combining AI-powered task management, productivity analytics, adaptive accessibility themes, and intelligent assistance into one seamless productivity platform.

## 📈 AI Progress Analysis

Instead of manually selecting completion percentages, users simply describe what they have completed.

Gemini AI analyzes:

- Task Title
- Task Description
- User Progress Update

and intelligently estimates:

- Progress Percentage
- Completion Analysis
- Remaining Work

## 🧠 Adaptive Workspace

A unique accessibility-focused feature that adapts the interface according to different cognitive and sensory preferences.

Available Modes

- 🎯 ADHD Focus Mode
- 🌿 Autism Calm Mode
- 🌙 Migraine Relief Mode
- 📖 Dyslexia Friendly Mode
- 🌑 Default Workspace

Each mode adjusts colors, typography, spacing, animations, and visual emphasis to provide a more comfortable and personalized productivity experience.
